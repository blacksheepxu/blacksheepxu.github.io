# Theme Code Change Checklist

- [ ] 这次改动是否触及共享壳层、首页、战区页、载体页或主题脚本？
- [ ] 是否阅读了受影响实现文件和已有相关测试？
- [ ] 是否确认 `content` capability 主题仍保持内容优先，不泄漏命令壳层交互？
- [ ] 是否确认 `command` capability 主题仍保持战区语义、交互状态和导航上下文？
- [ ] 如果改动触及 theme registry 或布局脚本，是否确认 theme id 与 capability 的映射仍正确？
- [ ] 如果改动触及热度、动态补位或首页状态条，是否补了对应单测或浏览器断言？
- [ ] 如果改动触及首页战区地图，是否确认固定核心战区坐标仍稳定、动态战区仍只在补位区内出现？
- [ ] 如果改动触及地图节点样式，是否确认热度颜色仍和 heat band 对应，而不是只剩装饰色？
- [ ] 是否在相关页面做了浏览器或截图验证？
- [ ] 是否需要更新 Playwright 断言？如果需要，是否已经更新？
- [ ] 如果跳过任何一项，是否已经写明原因？
