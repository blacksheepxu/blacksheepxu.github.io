# Theme Content Update Checklist

- [ ] 是否保持单一内容源，没有为某个主题单独维护内容版本？
- [ ] 这份内容是否在所有现有主题下都能正确显示？
- [ ] 如果内容会影响主题战区归属，是否检查了 strategy 首页与相关战区页？
- [ ] 如果内容落在 blog / research / lab 载体中，是否检查了相应 carrier page 和 detail page？
- [ ] 如果内容引入了新的长期主题线索，是否评估了核心战区、动态补位或外围散点是否需要调整？
- [ ] 如果内容填写或修改了 `updatedAt`、topic、focus、kind 等信号字段，是否确认首页热度和相关战区表现仍然合理？
- [ ] 如果内容变化影响了热度排序，是否确认首页地图中的节点颜色和动态补位位置没有出现明显失真？
