<h1 id="Tensor是什么？"><a href="#Tensor是什么？" class="headerlink" title="Tensor是什么？"></a>Tensor是什么？</h1><p>Tensor即张量，是一个多维数组，是标量、向量、矩阵的高维拓展。<br />标量是一个单个数，是一个0维张量；<br />向量则是多个标量在同一个维度上的组合，是一个1维张量；<br />矩阵是多个向量在同一个维度上的组合，是一个2维张量；<br />而多个矩阵组成的则是3维张量……依次类推，</p><table><thead><tr><th>名称</th><th>张量</th><th>说明</th></tr></thead><tbody><tr><td>标量</td><td>0维张量</td><td>单个数</td></tr><tr><td>向量</td><td>1维张量</td><td>多个标量在同一维度上的组合</td></tr><tr><td>矩阵</td><td>2维张量</td><td>多个向量在同一维度上的组合</td></tr><tr><td>……</td><td>……</td><td>……</td></tr><tr><td></td><td>n维张量</td><td>多个n-1维张量在同一维度上的组合</td></tr></tbody></table><p>我们正常见到的图片大部分就是3个矩阵，分别代表了相应位置的三原色rgb的数值(0~255)，组成了一张彩色的图片，这样三个矩阵的整体可以看做一个张量。  </p><h2 id="Tensor与Variable"><a href="#Tensor与Variable" class="headerlink" title="Tensor与Variable"></a>Tensor与Variable</h2><p>Variable是 torch0.4.0 版本之前专门列出来的一个torch.autograd中的一个数据类型，提出这个概念是为了方便计算梯度。  </p><p>Variable中包含如下属性：  </p><table><thead><tr><th>属性</th><th>作用</th></tr></thead><tbody><tr><td>data</td><td>张量的内容</td></tr><tr><td>grad</td><td>张量的计算梯度</td></tr><tr><td>grad_fn</td><td>张量使用的计算函数，方便计算梯度使用</td></tr><tr><td>requires_grad</td><td>该张量是否需要计算梯度</td></tr><tr><td>is_leaf</td><td>是否是叶子结点(计算图中使用)</td></tr></tbody></table><p>后来的pytorch中Variable并入Tensor，并且多出如下属性：<br />Tensor:  </p><table><thead><tr><th>属性</th><th>作用</th></tr></thead><tbody><tr><td>dtype</td><td>张量的数据类型，如torch.FloatTensor,torch.cuda.FloatTensor</td></tr><tr><td>shape</td><td>张量的形状，如(64,3,224,224)</td></tr><tr><td>device</td><td>张量所在的设备，GPU/CPU</td></tr></tbody></table><h1 id="Tensor的创建"><a href="#Tensor的创建" class="headerlink" title="Tensor的创建"></a>Tensor的创建</h1><h2 id="直接创建"><a href="#直接创建" class="headerlink" title="直接创建"></a>直接创建</h2><h3 id="从data创建"><a href="#从data创建" class="headerlink" title="从data创建"></a>从data创建</h3><p>创建命令：<code>torch.tensor()</code><br />功能：以data创建tensor  </p>

```python
torch.tensor(
data,# 数据，可以是list,numpy
dtype=None,# 数据类型，默认与data的一致(None即默认)
device=None,# 所在设备，cuda/cpu
requires_grad=False,# 是否需要梯度
pin_memory=False)# 是否存于锁页内存，与转换效率有关，默认为False
```

<h3 id="从numpy创建"><a href="#从numpy创建" class="headerlink" title="从numpy创建"></a>从numpy创建</h3><p>创建命令： <code>torch.from_numpy(ndarray)</code><br />功能： 从numpy创建tensor  </p><p>注意事项：用此方法创建的tensor与原ndarray<strong><em>共享内存</em></strong>，当修改其中一个的数据，另外一个也会被改动。  </p><h2 id="依据数值创建"><a href="#依据数值创建" class="headerlink" title="依据数值创建"></a>依据数值创建</h2><h3 id="zeros"><a href="#zeros" class="headerlink" title="zeros()"></a>zeros()</h3><p>创建命令：<code>torch.zeros()</code><br />功能：依据size创建全0张量</p><table><thead><tr><th>属性</th><th>内容</th></tr></thead><tbody><tr><td>size</td><td>张量的形状，如(3,3)、(3,224,224)</td></tr><tr><td>out</td><td>输出的张量</td></tr><tr><td>layout</td><td>内存中布局形式，有strided(默认),sparse_coo(稀疏张量时设置此可提高读取效率)等</td></tr><tr><td>device</td><td>所在设备</td></tr><tr><td>requires_grad</td><td>是否需要梯度</td></tr></tbody></table>

```python
torch.zeros(*size,
   out=None,
   dtype=None,
   layout=torch.strided,
   device=None,
   requires_grad=False)
```

<h3 id="zeros-like"><a href="#zeros-like" class="headerlink" title="zeros_like()"></a>zeros_like()</h3><p>创建命令：<code>torch.zeros_like()</code><br />功能：依input形状创建全0张量</p><table><thead><tr><th>属性</th><th>内容</th></tr></thead><tbody><tr><td>input</td><td>创建与input同形状的全0张量</td></tr><tr><td>dtype</td><td>数据类型</td></tr><tr><td>layout</td><td>内存中布局形式</td></tr></tbody></table>

```python
torch.zeros_like(input.
dtype=None,
layout=None,
device=None,
requires_grad=False)
```

<h3 id="ones-ones-like"><a href="#ones-ones-like" class="headerlink" title="ones,ones_like()"></a>ones,ones_like()</h3><p>用法同zeros和zeros_like()。</p>

```python
torch.ones(*size,
  out=None,
  dtype=None,
  layout=torch.strided,
  device=None,
  requires_grad=False)

torch.ones_like(input,
   dtype=None,
   layout=None,
   device=None,
   requires_grad=False)
```

<h3 id="full-full-like"><a href="#full-full-like" class="headerlink" title="full,full_like()"></a>full,full_like()</h3><p>用法同one和one_like()，只是可以指定填充值。<br />full():  </p><table><thead><tr><th>属性</th><th>内容</th></tr></thead><tbody><tr><td>size</td><td>张量的形状</td></tr><tr><td>fill_value</td><td>张量的填充值</td></tr></tbody></table>

```python
torch.full(size,
  fill_value,
  out=None,
  dtype=None,
  layout=torch.strided,
  device=None,
  requires_grad=False)
```

<h3 id="arrange"><a href="#arrange" class="headerlink" title="arrange()"></a>arrange()</h3><p>创建命令：<code>torch.arange()</code><br />功能：创建等差的一维张量<br />注意事项：数值区间为[start,end)</p><table><thead><tr><th>属性</th><th>说明</th></tr></thead><tbody><tr><td>start</td><td>数列起始值</td></tr><tr><td>end</td><td>数列”结束值”</td></tr><tr><td>step</td><td>数列公差，默认为1</td></tr></tbody></table>

```python
torch.arange(start=0,
end,
step=1,
out=None,
dtype=None,
layout=torch.strided,
device=None,
requires_grad=False)
```

<h3 id="linspace"><a href="#linspace" class="headerlink" title="linspace()"></a>linspace()</h3><p>创建命令：<code>torch.linspace()</code><br />功能：创建均分的1维张量<br />注意事项：数值区间为[start,end]</p><table><thead><tr><th>属性</th><th>说明</th></tr></thead><tbody><tr><td>start</td><td>数列起始值</td></tr><tr><td>end</td><td>数列结束值</td></tr><tr><td>steps</td><td>数列长度</td></tr></tbody></table>

```python
torch.linspace(start,
  end,
  steps=100,
  out=None,
  dtype=None,
  layout=torch.strided,
  device=None,
  requires_grad=False)
```
