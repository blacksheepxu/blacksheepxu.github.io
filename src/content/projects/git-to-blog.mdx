这个项目的目标，是把项目过程本身从“只存在于 AI 对话里”变成“可以回看、可以审核、可以继续发布”的长期资产。

它当前优先解决三件事：

1. 为每个项目维护一份持续更新的研究档案
2. 允许从 Codex / Claude Code 手动触发内容采集
3. 在真正写入站点之前，先经过本地审核和 diff 预览
